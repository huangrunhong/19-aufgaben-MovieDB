# Movie datenbank

Welcome! Movie datenbank provides you with thousands of movies.

### Function

- show all movies
- Show movie details
- search movie
- Collect movies
- View your favorite movies
- Edit movie
- Add new movie
- Register an account
- Verify account
- Login account
